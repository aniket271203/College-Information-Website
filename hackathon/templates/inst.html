Instructions To Use Campus Contacts

We're so glad you're here! Our website is a fun and safe place where you can learn new things, play games, and make friends. Here are some tips on how to use our site:

Step 1: To get around our website, you can use the navigation bar at the top of the page. This is where you'll find links to all of our different sections. Just click on a link to go to that section.

Home: Click on this link to go to the Home page that contains details about our team.

Faculty: Click on this link to go to the Faculty page that contains details about the faculty at IIIT-H.

Search: On this page we have the search option where you can input the labs or the name of the faculty you are looking for .

ADD New: To add a new faculty to database you click on this button but to execute the changes you need the authentication key.

Delete: To delete an existing faculty from database you click on this button but to execute the changes you need the authentication key.

Students: Click on this link to go to the Students page that contains details about the Students at IIIT-H.

Search: On this page we have the search option where you can input the roll number or the name of the student you are looking for .

ADD New: To add a new student to database you click on this button but to execute the changes you need the authentication key.

Delete To delete an existing student from database you click on this button but to execute the changes you need the authentication key.

View Details Clicking on the view details, the details of the students will be displayed as follows, also, the edit and grades option are dislpayed.

Edit details To edit the details of an existing student from database you click on this button but to execute the changes you need the authentication key.

View Grades To view the grades of an existing student from database you click on this button but to view the grades you need the authentication key.

Courses Courses in the navigation bar has two sections, as they are filtered accordingly into spring and monsoon.

Search On this page we have the search option where you can input the course code of the course you are looking for .
